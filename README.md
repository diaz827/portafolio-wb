## 📖 Descripción de la Práctica
Este proyecto consiste en el diseño y maquetación desde cero de un **currículum / portafolio web personal** estático e interactivo. La página está estructurada para presentar de forma clara, ordenada y atractiva mi perfil profesional, formación académica, experiencia laboral, habilidades blandas (*soft skills*) y los proyectos prácticos que he ido desarrollando.

### ✨ Características Principales
* **📱 Diseño 100% Responsivo:** Adaptación perfecta a cualquier dispositivo (móviles, tablets y pantallas de escritorio) gracias al uso estratégico de *Media Queries*.
* **🎨 Interactividad y UI/UX:** Transiciones suaves (*hover*) en los botones y tarjetas de las secciones, incluyendo efectos de escala y sombras dinámicas para una mejor experiencia de usuario.
* **📊 Gráficos de Progreso:** Implementación visual muy llamativa de barras de progreso circulares (tipo *donut*) para mostrar el nivel de conocimientos técnicos (HTML/CSS, Java, MySQL, etc.). ¡Todo hecho sin usar imágenes ni JavaScript!
* **🖼️ Galería Dinámica:** Sección de prácticas donde las capturas de pantalla de los proyectos se despliegan de forma fluida al pasar el cursor sobre ellas.

## 🛠️ Tecnologías Utilizadas
Para hacer realidad este proyecto, he utilizado las tecnologías base del desarrollo web Frontend:

* **🟠 HTML5:**
  * Estructuración del contenido mediante **marcado semántico** (`<header>`, `<main>`, `<section>`, `<nav>`, `<footer>`) para un código limpio, legible y accesible.

* **🔵 CSS3:**
  * **🎨 Variables Globales (`:root`):** Para mantener una paleta de colores coherente y facilitar el mantenimiento del código a futuro.
  * **📏 Flexbox:** Como motor principal de la maquetación, logrando una distribución fluida de columnas y un centrado perfecto de los elementos.
  * **🍩 Conic-Gradients:** Uso avanzado de la propiedad `conic-gradient` para dibujar los gráficos circulares de progreso de forma nativa.
  * **🎬 Animaciones y Efectos:** Uso intensivo de propiedades como `transition`, `transform` y `box-shadow` para dotar de vida y dinamismo a la interfaz.
